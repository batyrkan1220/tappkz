Список методов
Базовые методы
Операция

Описание

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/offers/calculate

Возвращает доступные варианты доставки.

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/create

Cоздает заявку с переданными параметрами. Заявка попадает в систему логистики и запускает процесс заказа.

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/info

Возвращает основную информацию по заявке.

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/accept

Подтверждает заявку при успешной оценке. После подтверждения заявка перейдет в статус accepted и сервис запустит процесс поиска исполнителя.

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/cancel-info

Возвращает информацию об условиях отмены.

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/cancel

Отменяет заявку, которая была подтверждена. Операция может быть выполнена в течение ограниченного времени.

Дополнительные методы
1. Стоимость и тарифы
Операция

Описание

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/check-price

Возвращает первичную оценку стоимости без создания заявки.

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/tariffs

Возвращает тарифы, доступные в точке.

2. Отслеживание заявки
Операция

Описание

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/driver-voiceforwarding

Возвращает номер телефона для звонка курьеру, который выполняет заявку в настоящий момент.

GET /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/performer-position

Возвращает координаты, скорость и направления движения исполнителя заявки.

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/points-eta

Возвращает список точек и оценку времени прибытия на точку курьером.

GET /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/tracking-links

Возвращает ссылки для отслеживания курьера.

3. Код подтверждения
POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/confirmation_code

Возвращает код подтверждения в текущей точке (если это возможно).

4. Подтверждение доставки
POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/proof-of-delivery/info

Возвращает данные, подтверждающие доставку.

5. Редактирование заказа
Операция

Описание

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/edit

Изменяет параметры заявки. Операция доступна до принятия оффера клиентом.

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/apply-changes/request

Частичное редактирование заявки после ее подтверждения.

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/apply-changes/result

Получает результат применения изменений.

POST /b2b.taxi.yandex.net/b2b/cargo/integration/v2/claims/return

Инициирует возврат заказа курьером в точку получения.
